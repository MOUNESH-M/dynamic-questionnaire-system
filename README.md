# Dynamic Questionnaire System

## Tech Stack

Frontend:
- React
- Axios

Backend:
- FastAPI
- JWT Authentication

Database:
- MongoDB

## Features

- User Authentication
- Questionnaire Management
- Question Management
- Option Management
- Response Submission

## Setup

### Backend

pip install -r requirements.txt

uvicorn app.main:app --reload

### Frontend

npm install

npm run dev
